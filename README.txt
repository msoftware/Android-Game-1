ACCELERO-MAZE:-

Accelero-maze is a primarily a motion sensor game that makes use of the accelerometer sensor in the mobile device. Accelerometer sensor basically detects the tilt in the device and thus processes the running application. Accelero-maze is envisaged to be a 2D motion sensor game that consists of an object such as a ball that rolls in the direction of the tilt. The game features obstacles that have to be dodged off and certain objects such as coins that have to be fetched. The object finally has to be maneuvered to the exit. 

The game is aimed at people who love playing motion sensor games that have taken the gaming experience to a whole new level in the recent times.
